# Population Change and Urban Density in New York City (1970 - 2010)

This project presents a GIS-based spatial analysis of population change and urban density across New York City community districts between 1970 and 2010.

## Overview
The analysis compares:
- Percent population change
- Absolute population change
- Population density (people per square mile)

A triptych map layout is used to enable direct spatial comparison of demographic trends across boroughs.
